# Assets

Place Figma-exported assets here (e.g. SpotsNow logo, QCode logo, stat icons, search icon, filter icon, listing placeholders). The app uses placeholder images and inline SVGs until these are added.

Recommended filenames:
- `spotsnow-logo.svg` (or .png)
- `qcode-logo.png` (112×112)
- `icon-impressions.svg`, `icon-clicks.svg`, `icon-users.svg`, `icon-trend.svg`, `icon-dollar.svg`
- `icon-search.svg`, `icon-filter.svg`
